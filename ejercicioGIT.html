Examen Git y Github

Alumn@: J.A.M

  1. ¿Que es un sistema de control de versiones?

    [Es una herramienta que ayuda a los equipos de desarrollo de software a gestionar los cambios del codigo fuente a lo largo del tiempo]

  2. ¿Cual es la diferencia entre git y github?

    [Git es un sistema de control de versiones distribuido y github es una plataforma de alojamiento de codigo basada en la web que usa Git]

  3. ¿Cómo creo una rama en git?

    [Para crear una rama debemos de usar en la terminal el comando "git branch"]

  4. ¿Cómo clono un repositorio?

    [Para clonar un repositorio debemos usar en la terminal el comando "git clone [repositorio]"]

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    [Para hacerlo deberas usar en la terminal el comando "git push"]

  6. ¿Cómo puedo generar un tag?

    [Deberemos de poner en la terminal el comnado "git tag"]

  7. Cómo hago una mezcla de ramas?

    [Debes usar en la terminal el comando "git merge [nombre-rama]"]

  8. ¿Cómo puedo moverme a un commit especifico?

    [Usaremos el comando git checkout en la terminal]

  9. ¿Cómo puedo borrar una rama?

    [Debes poner en la terminal el comando "git push origin --delete [rama]"]

  10. ¿Cómo puedo borrar un tag?

    [Debes de poner en la terminal el comando "git push origin --delete [nombre del tag]]

  11. ¿Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [Con el comando "git log origin/main..HEAD]

  12. ¿Para que sirve el comando git remote -v?

    [Sirve para ver a que repositorios remotos esta conectado a tu proyecto y que URLS usa para enviar y recibir]

  13. ¿Cómo le digo a git cual es mi email y correo?

    [git config --globar user.name y git config --global user.email]
